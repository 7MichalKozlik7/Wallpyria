## Overview

**Wallpyria** transforms your multi-monitor macOS setup with stunning wallpapers. Create, manage, and schedule static, animated, and dynamic wallpapers across all your displays with ease.

Whether you have a dual-monitor workstation or a complex multi-display setup, Wallpyria gives you complete control over your desktop aesthetics.

## Features

### Wallpaper Modes

| Mode | Description |
|------|-------------|
| **Static** | Single image, different per monitor, or panorama across displays |
| **Animated** | Video wallpapers (.mov, .mp4) with smooth playback |
| **Dynamic HEIC** | Time-based, solar position, or appearance-based wallpapers |
| **Scheduled** | Automatic wallpaper rotation based on time of day |

### Key Capabilities

- **Multi-Monitor Support** — Automatic detection and individual configuration for each display
- **Panorama Mode** — Stretch wallpapers seamlessly across multiple monitors
- **Smart Scheduling** — Schedule different wallpapers for different times of day
- **Dynamic Wallpapers** — Create and edit Dynamic HEIC wallpapers with visual timeline editor
- **ML Upscaling** — Machine learning image enhancement for lower resolution images
- **Quality Presets** — Native, High, Super High, and Ultra quality settings
- **Video Wallpapers** — Support for .mov and .mp4 animated backgrounds
- **9 Languages** — English, Polish, German, French, Spanish, Italian, Russian, Japanese, Chinese

## Website

This repository contains the source code for the official Wallpyria website at [wallpyria.app](https://wallpyria.app).

### Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **i18n**: next-intl

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Project Structure

```
wallpyria-website/
├── src/
│   ├── app/[locale]/     # Internationalized pages
│   ├── components/       # React components
│   └── i18n/             # i18n configuration
├── messages/             # Translation files (9 languages)
├── public/               # Static assets & screenshots
└── out/                  # Static export (production build)
```
